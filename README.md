# QuantDesk 📊
A live quant-style dashboard for the top 10 US tech stocks, built with Python, Dash, and Yahoo Finance.

## Features
- Live price data via Yahoo Finance
- Monte Carlo simulation (500 paths, 63-day horizon)
- Correlation matrix & risk-return scatter
- Rolling volatility, VaR & CVaR (95%)
- Sharpe, Sortino, Calmar ratios
- Quant scoring per stock

## Install & Run
pip install -r requirements.txt
python quantdesk.py

Then open http://127.0.0.1:8050
